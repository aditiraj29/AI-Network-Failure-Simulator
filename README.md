# AI-Network-Failure-Simulator

An interactive AI-powered simulator that models cascading failures in complex networks.

## Features
- AI generated failure scenarios
- Cascade failure visualization
- Critical node detection
- Network damage analysis
- AI recommendations for preventing network collapse

## Tech Stack
Python
Gradio
NetworkX
Matplotlib

## Demo
Interactive web app that simulates cascading failures in power grids, computer networks, and transport systems.
